excel-worker — Quick Start

A lightweight Java 21+ library for reading .xlsx files: sheets, rows, and typed cell values (text, number, boolean, empty).

On this page Introduction Dependency Code examples cellValue() Limitations

Introduction

Add the library to a JVM project (Gradle or Maven), then open a workbook with Xlsx.read(Path) and iterate sheets, rows, and cells. Pick a language tab below for a minimal example.

Dependency (JitPack)

Declare the JitPack repository and this coordinate (replace the version with a tag or commit from the project):

com.github.minpor:excel-worker:v0.1.4

Code examples

Same API on the JVM — choose a tab to see the pattern in your language.

import io.github.minpor.excel.*;
import java.nio.file.Path;

XlsxWorkbook wb = Xlsx.read(Path.of("report.xlsx"));
for (XlsxSheet sheet : wb.sheets()) {
    for (XlsxRow row : sheet.rows()) {
        for (var cell : row.cellsByColumn().values()) {
            switch (cell.value()) {
                case CellValue.Empty e -> {}
                case CellValue.Text t -> System.out.println(t.value());
                case CellValue.NumberValue n -> System.out.println(n.value());
                case CellValue.BooleanValue b -> System.out.println(b.value());
            }
        }
    }
}

import java.nio.file.Path

def wb = Xlsx.read(Path.of("report.xlsx"))
wb.sheets().each { sheet ->
    sheet.rows().each { row ->
        row.cellsByColumn().values().each { cell ->
            def v = cell.value()
            if (v instanceof CellValue.Text) println(v.value())
            else if (v instanceof CellValue.NumberValue) println(v.value())
            else if (v instanceof CellValue.BooleanValue) println(v.value())
        }
    }
}
import io.github.minpor.excel.*
import java.nio.file.Path
import scala.jdk.CollectionConverters.*

val wb = Xlsx.read(Path.of("report.xlsx"))
for
  sheet <- wb.sheets().asScala
  row <- sheet.rows().asScala
  cell <- row.cellsByColumn().values().asScala
do
  cell.value() match
    case t: CellValue.Text => println(t.value())
    case n: CellValue.NumberValue => println(n.value())
    case b: CellValue.BooleanValue => println(b.value())
    case _: CellValue.Empty => ()
(import '[io.github.minpor.excel Xlsx CellValue]
        '[java.nio.file Path])

(def wb (Xlsx/read (Path/of "report.xlsx")))
(doseq [sheet (.sheets wb)
        row (.rows sheet)
        cell (.values (.cellsByColumn row))]
  (let [v (.value cell)]
    (cond
      (instance? CellValue$Text v) (println (.value ^CellValue$Text v))
      (instance? CellValue$NumberValue v) (println (.value ^CellValue$NumberValue v))
      (instance? CellValue$BooleanValue v) (println (.value ^CellValue$BooleanValue v))
      :else nil)))

Fixed columns: cellValue()

Sparse rows do not use null for missing columns: row.cellValue(index) returns CellValue.EMPTY. Columns are 0-based; ExcelColumns.columnIndex maps A→0, B→1, … CellValues reads text/number/boolean without coercion.

import io.github.minpor.excel.*;
import java.nio.file.Path;

XlsxWorkbook wb = Xlsx.read(Path.of("report.xlsx"));
XlsxSheet sheet = wb.sheet("Summary").orElseThrow();

for (XlsxRow row : sheet.rows()) {
    String name = CellValues.asText(row.cellValue(0)).orElse("");
    double amount = CellValues.asNumber(row.cellValue(ExcelColumns.columnIndex("C"))).orElse(0.0);
}

(import '[io.github.minpor.excel Xlsx CellValues ExcelColumns]
        '[java.nio.file Path])

(let [wb (Xlsx/read (Path/of "report.xlsx"))
      sheet (.orElseThrow (.sheet wb "Summary"))]
  (doseq [row (.rows sheet)]
    (let [name (.orElse (CellValues/asText (.cellValue row 0)) "")
          amount (.orElse (CellValues/asNumber
                             (.cellValue row (ExcelColumns/columnIndex "C")))
                    0.0)]
      (println name amount))))

Current limitations

  • .xlsx only; legacy .xls is not supported.
  • Read-only — writing workbooks is not supported.
  • Date/time values may appear as numeric serials (CellValue.NumberValue).
